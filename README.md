# GS-Plaza Business Center Website

## Overview
GS-Plaza is a modern business center website built with HTML, CSS, and JavaScript. The site showcases real estate properties with a focus on luxury apartments and business facilities.

## Features
- Responsive design optimized for all devices
- Interactive hero slider with parallax effect
- Property listings with detailed descriptions
- Gallery section with image carousel
- Contact forms with validation
- Animated counters and transitions
- Mobile-friendly navigation

## Technology Stack
- HTML5
- CSS3
- JavaScript/jQuery
- Bootstrap framework
- Owl Carousel for sliders
- Custom animations and effects

## Installation
1. Clone the repository
2. Open `index.html` in a web browser
3. No build tools or dependencies required

## Customization
The website can be easily customized by modifying:

- JavaScript behaviors in custom.js
