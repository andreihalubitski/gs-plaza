Universal Classes

These classes gives you ability to style some elements in some conditions without writing a CSS code. For example, if you want colored font on dark background (because the font color on dark background is automatically set to white) you can use class text-color-white. Real example of use can be found further in the documentation. Here is the list of universal classes:

  • center use for Texts Centers the text
  • no-margin use for Any Remove margin of element
  • no-bottom-padding use for block Removes bottom padding of the block
  • no-padding use for block Removes whole padding of the block
  • move-up use for Any Moves element 20px up
  • text-align-right use for Texts Align text to right
  • bg-expand-right use for block Expands background to the right
  • bg-expand-left use for block Expands background to the left
  • bg-expand-both use for block Expands background to the both sides
  • bg-dark use for block Sets background color to dark grey
  • bg-default use for block Sets background color to default color of scheme
  • bg-light use for block Sets background color to light grey
  • bg-white use for block Sets background color to white
  • bg-opacity-5 use for block Sets background opacity to 5%
  • bg-opacity-10 use for block Sets background opacity to 10%
  • bg-opacity-20 use for block
  • bg-opacity-30 use for block
  • bg-opacity-40 use for block
  • bg-opacity-50 use for block
  • bg-opacity-60 use for block
  • bg-opacity-70 use for block
  • bg-opacity-80 use for block
  • bg-opacity-90 use for block Sets background opacity to 90%
  • half-margin use for block Cuts the margin top and bottom to half
  • va-middle use for block Vertical align element inside the block
  • text-color-white use for Texts Sets color of font to white
  • text-color-dark use for Texts Sets color of font to dark grey
  • text-color-light use for Texts Sets color of font to light grey
  • text-color-default use for Texts Sets color of font to default color

How to change the color scheme

There are several color schemes prepared for your needs. If you want to use one from these color schemes, you need to change the name of desired CSS file in the <head> tag. For example you want to change the default color (which is red) to lime green:

Change the 
<link rel="stylesheet" href="assets/css/style.css" type="text/css">
Into 
<link rel="stylesheet" href="assets/css/color-lime-green.css" type="text/css">

Here is the list of predefined schemes:

  • color-blue
  • color-cyan
  • color-deep-magenta
  • color-light-brown
  • color-lime-green
  • color-orange
  • color-red-medium
  • color-red
  • color-royal-blue
